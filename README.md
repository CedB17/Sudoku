# Sudoku

Jeu de Sudoku en HTML/CSS/JavaScript avec interface graphique.

## Fonctionnalités

- Génération aléatoire d'une grille valide.
- Cinq niveaux de difficulté : **très facile**, **facile**, **moyen**, **difficile**, **expert**.
- Vérification de la grille remplie.
- Affichage de la solution.

## Lancer le projet

Comme il s'agit d'une application statique, vous pouvez simplement ouvrir `index.html` dans votre navigateur.

Option serveur local (recommandé) :

```bash
python -m http.server 8000
```

Puis ouvrir : `http://localhost:8000`.
